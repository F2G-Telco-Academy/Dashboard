# Telecom Drive Test Dashboard — Landing Page

Professional landing page for the **Telecom Drive Test Dashboard** by F2G-SOLUTIONS / PKFOKAM48 - TELCO ACADEMY.

## Stack

- **Vite** + **React 19** + **TypeScript**
- **Tailwind CSS 3** — dark telecom theme (`#698D9C` accent, `#0F1923` background)
- **Framer Motion** — scroll animations
- **Lucide React** — icons

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
npm run preview
```

Deploy to Netlify — config already included in `netlify.toml`.

## Sections

- **Hero** — product intro with RAT badges and dashboard preview
- **Features** — 12 key capabilities in a responsive grid
- **How It Works** — 4-step timeline (Upload → Customize → Analyze → Share)
- **Use Cases** — Network Engineers, Field Teams, Optimization, Client Presentations
- **FAQ** — Accordion with common questions
